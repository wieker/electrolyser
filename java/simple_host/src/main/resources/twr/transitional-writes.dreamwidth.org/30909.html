Ну да, LVDS компаратор,
массив LVDS компараторов + программируемая логика + элементарный сенсор <= это все, что вам
нужно знать про физические приборы, про компьютеры, про человеческий мозг с т.з. физика.

LVDS компаратор - базовый блок любого ADC.

в физике априорные тесты, не сводимые к самой физике:

1. оружие. ебанет/не ебанет

2. делать деньги, прибыльно неприбыльно. LVDS компаратор

3. красивая картинка. девушек пофоткать не шибко одетых на цифровой фотоаппарат.
привлекает/не привлекает.

все остальное будет рекурсией, ссылкой на саму физику.

в госплане, как социоинженерной структурой, созданной физикой по образу
физического прибора все то же самое.
как структуре созданной по подобию программы - то же самое.

нужны тесты. тесты false/true.
больше/меньше. LVDS компаратор.
это же физика. это же компьютер.
LVDS компаратор миски риса.
прибыль-убыток.

больше там ничего нет.

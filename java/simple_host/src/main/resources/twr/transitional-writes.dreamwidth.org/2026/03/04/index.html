сколь угодно сложного положения.

Пойа, колмогоровская сложность,
Бастиа, элементарная эротика,
симметрия всех людей относительно
Бастиа.

Выводите все сами из априорных принципов.
Возьмите православный сталинизм
и/или сталинизм и/или исламизм
и/или сатанизм и выведете
все сами.
Симметрия, элементарная физика,
красивые девушки (для девушек тут дети?).

1. хлоратно-битумный детонатор и прочее оружие из первых принципов
2. эротические фотосессии и прочие полезные рабочие советы и что вам мешает.
3. выведите калькуляционный аргумент Мизеса.
4. из симметрии выводите этику. дети, проповеди, революции.
5. будьте за прогресс
